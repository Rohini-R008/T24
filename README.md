# Animations & UI Feedback using Tailwind CSS

## Overview

In this project, I created a simple webpage to demonstrate different types of animations and UI interactions using **Tailwind CSS**. The main goal was to understand how animations can improve user experience by giving visual feedback and making the interface more interactive.

## Features

* I implemented **built-in Tailwind animations** like spin, ping, pulse, and bounce using small demo cards.
* Created different **button animations** such as scaling, rotation, color change, and combined hover effects.
* Designed **interactive cards** with hover effects like lift, shadow, and border highlight.
* Added **notification badges** with small indicators to show alerts.
* Included different **loading states** like skeleton loaders, spinner buttons, and bouncing dots.
* Implemented **staggered animations** using delays to show smooth appearance of elements.

## Technologies Used

* HTML
* Tailwind CSS (CDN)
* JavaScript (for basic interaction)

## How to Run

Open the `index.html` file in a browser. Internet connection is required for Tailwind CSS.

## What i learnt

Through this project, I learned how to use Tailwind animation utilities, transitions, and keyframes to create interactive UI elements without writing complex CSS.
